File Format Instructions

Upload a CSV or Excel file containing the following columns:

Patient Information
  • name - Patient's full name
  • phone - Patient's phone number
  • address - Patient's street address
  • city - Patient's city
  • state - Patient's state (2-letter code)
  • zip - Patient's ZIP code
  • dob - Date of birth (YYYY-MM-DD format)
  • medicare - Medicare information
PCP Information
  • pcp_name - Primary Care Physician name
  • pcp_address - PCP office address
  • pcp_city - PCP city
  • pcp_state - PCP state (2-letter code)
  • pcp_zip - PCP ZIP code
  • pcp_phone - PCP phone number
  • pcp_fax - PCP fax number
  • pcp_npi - PCP NPI number
Note: The system will generate fax documents based on the selected device type for all records in your file. Make sure your CSV/Excel file has the exact column names shown above.
